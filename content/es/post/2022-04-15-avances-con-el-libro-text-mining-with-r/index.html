Today I finished Chapter 4 of the book Text Mining with R by Julia Silge and David Robinson. Since my goal is to serve as practice, I not only include the code but also the comments and the theoretical proposal. In some cases, I also include information about functions or terms that I did not know before.

Each chapter is a rmarkdown file in my github repo:

  • Chapter 1: tidy-text format

  • Chapter 2: Sentiment Analysis

  • Chapter 3: Word Frequency and Document Frequency Analysis with td-idf

  • Chapter 4: Relations between words with n-grams and correlations

I hope that these materials will be useful to someone who wants to learn about text analysis, but is not so comfortable with English. If you need a hand, I’m happy to help.
